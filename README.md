# This is an Chip 8 Emulator for the PlayStation 2 :D

## Is very simple, but functional :)

# Input

| **Chip 8 Keypad** | **Buttons Dualshock 2**   |
|------------------|---------------------------|
| 0                | START                     |
| 1                | L1                        |
| 2                | R1                        |
| 3                | L2                        |
| 4                | TRIANGLE                  |
| 5                | UP                        |
| 6                | SQUARE                    |
| 7                | LEFT                      |
| 8                | DOWN                      |
| 9                | RIGHT                     |
| A                | SELECT                    |
| B                | L3                        |
| C                | R2                        |
| D                | CIRCLE                    |
| E                | CROSS                     |
| F                | R3                        |
